# Python TCP Port Scanner

## Overview
A multi-threaded TCP port scanner built in Python using the socket module.

## Features
- Scans common ports
- Multi-threaded for faster execution
- Service name detection
- Execution time measurement

## Technologies Used
- Python 3
- socket
- threading

## Usage
python scanner.py
